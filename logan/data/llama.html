Question: Question: How many cheese balls are in a 35oz barrel if the 24oz barrel has 60 servings and each serving has 12 cheese balls?
Reformulated Question: If a 24oz barrel has 60 servings, and each serving has 12 cheese balls, how many cheese balls are in a 35oz barrel?

Answer:To solve this problem, we first need to calculate the number of cheese balls per ounce. Since there are 60 servings with 12 cheese balls each in a 24oz barrel, the total number of cheese balls is 60*12 = 720. So, for every oz, there are 720/24oz = 30 cheese balls per oz. Since we need to find out how many cheese balls are in a 35oz barrel, we multiply the number of cheese balls per oz by the size of the new barrel: 35oz * 30 cheese balls/oz = 1050 cheese balls. The answer is 1050.
